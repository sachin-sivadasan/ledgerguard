---
title: 'Get Usage by GID'
api: 'GET /v1/usages/{shopify_gid}'
description: 'Retrieve billing status for a usage charge'
---

## Path Parameters

<ParamField path="shopify_gid" type="string" required>
  The Shopify GraphQL ID for the usage record.

  Example: `gid://shopify/AppUsageRecord/67890`
</ParamField>

## Response

<ResponseField name="usageId" type="string" required>
  The Shopify GraphQL ID
</ResponseField>

<ResponseField name="billed" type="boolean" required>
  Whether Shopify has billed this charge
</ResponseField>

<ResponseField name="billingDate" type="datetime">
  When the charge was billed (null if not billed)
</ResponseField>

<ResponseField name="amountCents" type="integer" required>
  Charge amount in cents
</ResponseField>

<ResponseField name="description" type="string">
  Usage charge description
</ResponseField>

<ResponseField name="subscription" type="object">
  Parent subscription summary with payment status
</ResponseField>

<RequestExample>
```bash cURL
curl https://api.ledgerguard.app/v1/usages/gid://shopify/AppUsageRecord/67890 \
  -H "X-API-Key: lg_live_xxxxxxxxxxxx"
```

```javascript Node.js
const response = await fetch(
  'https://api.ledgerguard.app/v1/usages/gid://shopify/AppUsageRecord/67890',

);
```

```python Python
import requests
import os

response = requests.get(
    'https://api.ledgerguard.app/v1/usages/gid://shopify/AppUsageRecord/67890',
    headers={'X-API-Key': os.environ['LEDGERGUARD_API_KEY']}

data = response.json()
```
